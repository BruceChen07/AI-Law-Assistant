# 连续滚动预览实施里程碑（分阶段）

## M0 评审冻结
- 输出：技术方案、原型稿、基线报告评审通过
- 准入：确认灰度开关与回滚策略

## M1 架构改造（不改接口）
- 内容：
  - 新增连续滚动组件骨架
  - 引入虚拟窗口与页锚点状态
- 验收：
  - 功能开关可切换新旧预览
  - 基础连续阅读可用

## M2 性能优化
- 内容：
  - IntersectionObserver 懒加载
  - 预取调度器与请求取消
  - ObjectURL LRU 回收池
- 验收：
  - 100页文档流畅滚动
  - 内存峰值受控

## M3 UI/UX 完成
- 内容：
  - 页码浮标、进度条、缩放控制
  - 风险定位高亮动画
  - 移动端抽屉式缩略条
- 验收：
  - 桌面/移动体验一致
  - 可用性评审通过

## M4 灰度发布
- 内容：
  - 10% 管理员灰度 -> 30% 用户 -> 全量
  - 实时观测与告警
- 验收：
  - 关键指标达标
  - 无P1故障

## M5 收尾与沉淀
- 内容：
  - 清理旧分页技术债
  - 补齐测试与操作手册
- 验收：
  - 代码合并主干
  - 回归测试全通过

## 回滚机制
- 任何阶段出现严重问题：
  - 关闭 `preview_continuous_enabled`
  - 前端即时回退旧分页预览
  - 后端接口保持兼容，无需数据回迁
